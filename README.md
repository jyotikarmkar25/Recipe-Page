# Recipe Page

Welcome to my personal recipe book! This project is a simple, easy-to-navigate collection of my favorite recipes, including classic Indian dishes like Dal Chawal, Aalu Paratha, and Kheer.

## 🍽️ Recipes Included

- **Dal Chawal**: A comforting staple of lentil soup paired with steamed rice.
- **Aalu Paratha**: Delicious wheat flatbreads stuffed with a savory spiced potato filling.
- **Kheer**: A creamy and sweet traditional Indian rice pudding.

## ✨ Features

- **Easy Navigation**: Simple links on the home page to access each recipe.
- **Detailed Instructions**: Each recipe includes a list of ingredients and step-by-step cooking instructions.
- **Mobile Friendly**: Basic responsive design for viewing on different devices.

## 🛠️ Technologies Used

- **HTML5**: For the structure of the recipe pages.
- **CSS**: (Basic styling via HTML tags)
- **Local Images**: Visual representations of the delicious dishes.

## 🚀 How to Run

1. Clone or download this repository to your local machine.
2. Open `index.html` in any modern web browser (Chrome, Firefox, Safari, etc.).
3. Click on the recipe links to explore how to make each dish!
